## Third-party publication condition

The public patterns use the locally frozen Swiss-lever reference asset in
`src/ontology_watch_generator/third_party/grabcad/swiss_lever_watch_escapement/`.
The original model is
*Swiss Lever Watch Escapement Model* by David Velez. The original page,
download URL recovered from the ZIP metadata, exact file hashes and the
project's geometric modifications are recorded in `THIRD_PARTY_ASSETS.md`.

The release includes this modified integration under the project's explicit
attribution decision. The attribution notice must remain with every
redistribution; the project does not claim authorship of the source model.
